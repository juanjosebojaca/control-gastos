★ GUÍA PASO A PASO — ATAJOS iOS
PASO 1 — Crear el Atajo
Abre Atajos en tu iPhone → toca "+" arriba a la derecha → nómbralo "Control Gastos Auto"
PASO 2 — Agregar acción JavaScript
Toca "Agregar acción" → busca "Ejecutar JavaScript en Safari" → agrégala. Luego pega este código exacto: // ── PARSER UNIVERSAL WALLET iOS ────────────────── // Funciona con TODAS las tarjetas de Apple Pay // Formato Wallet: "Visa Infinite — $199.900 en Bershka" // o: "Mastercard ••5199 — $75.080 en Fatelares" var txt = input; // Texto de la notificación Wallet // Extraer monto — busca patrón $199.900 o $ 199.900 var montoMatch = txt.match(/\$\s*([\d][.\d]+|\d+)/); var montoRaw = montoMatch ? montoMatch[1].replace(/\./g,'') : '0'; var monto = parseInt(montoRaw) || 0; // Extraer comercio — texto después de "en " al final var comercioMatch = txt.match(/\ben\s+(.+?)(?:\s*$|\s*,)/i); var comercio = comercioMatch ? comercioMatch[1].trim() : 'Comercio Apple Pay'; // Extraer tarjeta — texto antes del " — " var tarjetaMatch = txt.match(/^(.+?)\s*[—–-]/); var tarjeta = tarjetaMatch ? tarjetaMatch[1].trim() : ''; // Fecha de hoy var hoy = new Date().toISOString().slice(0,10); var mk = hoy.slice(0,7); // Guardar en la app if (monto > 0) { var txs = JSON.parse(localStorage.getItem('cdg_main') || '[]'); txs.unshift({ id: Date.now(), type: 'expense', category: 'otros', amount: monto, note: comercio, card: tarjeta, date: hoy, mk: mk, auto: true }); localStorage.setItem('cdg_main', JSON.stringify(txs)); completion('✅ ' + comercio + ' · ' + monto.toLocaleString('es-CO') + ' · ' + tarjeta); } else { completion('⚠️ No se pudo leer el monto: ' + txt); }
PASO 3 — Conectar la Automatización
Ve a la pestaña Automatización (abajo en Atajos) → toca "+" → "Crear automatización personal" → busca "Transacción con Apple Pay" → selecciónala
PASO 4 — Configurar el trigger
En la automatización → "Con cualquier tarjeta" (cubre tus 4 tarjetas automáticamente) → toca "Siguiente" → agrega acción → "Ejecutar atajo" → selecciona "Control Gastos Auto"
PASO 5 — Sin confirmación
Activa "Ejecutar inmediatamente" (sin pedir confirmación cada vez) → toca "Listo". ¡Ya está configurado!
PASO 6 — Abrir la app en Safari
Abre Control de Gastos en Safari y déjala en segundo plano. Cada pago con Apple Pay se registrará automáticamente con badge AUTO y la tarjeta usada.

⚠️ Importante: La automatización "Transacción con Apple Pay" es el trigger más directo — es nativa de iOS y se activa en el momento exacto del pago, antes de cualquier notificación bancaria.
💡 NOTA SOBRE EL TEXTO DISPONIBLE
En la automatización "Transacción con Apple Pay", el Atajo recibe automáticamente: Nombre del comercio, Monto, Tarjeta usada y Fecha. Puedes pasar estas variables directamente como parámetros al script sin necesidad de parsear texto.
